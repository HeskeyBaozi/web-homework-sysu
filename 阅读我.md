## 如何启动？

在根目录下打开终端输入
```bash
$ npm start
```
该命令会启动服务器

## 说明
S2 - S5共用了S1的代码。
S2-S5也具有S1的功能。
CSS中@按钮的范围就是那一个圆圈，鼠标移出来就马上收回去了。所以只要鼠标进入@+便展开彩蛋，再进入便收回。
修复小按钮也能发起请求的bug。
